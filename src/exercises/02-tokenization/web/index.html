02 · Tokenization

One vocabulary, four languages

A single 10,000-token BPE vocabulary shared across India's Wikipedia article in English, Hindi, Telugu, and Tamil. The score rewards tokenizing all four equally efficiently: 1000 / (X₄ − X₁), where X is tokens-per-word. Flip between tokenizers and inspect every token.

Built with a hand-rolled pipeline (HuggingFace tokenizers), no runtime dependencies.
