1

Activations exist for a reason

A model with no nonlinearity can only draw a straight boundary — so it cannot separate two concentric rings. Add a single ReLU hidden layer and it can. Only the activation changes.

2

Depth without nonlinearity is a lie

Five stacked linear layers collapse to a single linear map — so they're no stronger than one layer, and fail the ring identically. Drop ReLUs between the same five and it's solved.

3

Embeddings learn similarity from nothing but next-token

Trained only to predict the next token in a tiny grammar, the embedding table clusters related words — even though similarity was never supplied. Watch animals, fruits, and verbs separate.

4

Memorization vs generalization — data closes the gap

A high-capacity net on tiny data drives train accuracy to ~100% while test accuracy lags — it memorizes. Grow the dataset and the train→test gap collapses.
