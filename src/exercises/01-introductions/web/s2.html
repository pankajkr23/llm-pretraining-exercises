02 · Depth

Five maps, one matrix

Stacking layers feels like adding power. But a chain of linear maps x → W₁x → W₂(W₁x) → … is just one linear map. Add depth: the transform gets no richer, and the whole stack collapses into a single matrix M. Only a nonlinearity in between can make depth mean anything — flip to ReLU and the collapse breaks.
