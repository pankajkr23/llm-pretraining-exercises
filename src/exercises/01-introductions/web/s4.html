04 · Data

Memorise, or generalise

The same over-parameterized network, trained on more and more data. On a tiny set it drives training accuracy to 100% by memorising every point — noise and all — while a held-out test set lags far behind. Drag the dataset size up and watch the jagged, memorised boundary relax into a smooth one as the train→test gap closes. Data is the regulariser.
